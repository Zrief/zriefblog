# Quartz 部署流程

## 📦 前置环境准备
1. 安装必备工具：
   - Git（[官网下载](https://git-scm.com/)）
   - Obsidian（[官网下载](https://obsidian.md/)）
   - Node.js v20+（[官网下载](https://nodejs.org/)）
   > 💡 提示：安装完成后若提示"找不到命令"，请重启系统刷新环境变量

## 🚀 Quartz 初始化
```bash
# 克隆 Quartz 仓库
git clone https://github.com/jackyzha0/quartz.git

# 进入项目目录
cd quartz

# 安装依赖
npm install

# 初始化配置
npx quartz create
````

## ⚙️ Git 配置

- 设置远程仓库源：
```
git remote set-url origin <你的仓库URL>
```
- 拉取源文件到本地仓库

## 🔧 GitHub 工作流设置

1. 编辑部署配置文件：  
    `quartz/.github/workflows/deploy.yml`
    > 📌 注意：`quartz` 是项目的根目录路径
2. 确保工作流文件包含正确的构建和部署配置

## 🛠️ 常用指令

| 命令                         | 功能                        |
| -------------------------- | ------------------------- |
| `npx quartz sync`          | 同步 GitHub 仓库内容并更新网页       |
| `npx quartz build --serve` | 本地构建并启动实时预览服务器（默认端口：8080） |
