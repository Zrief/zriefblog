摘要和结果都比较容易，以一句一句为单元。
讲不了结果展示的通用逻辑。
引言的目的，是引言的通用性。

如何进行引言的素材积累。
1. 阅读和做笔记的常见问题
2. 区分文章中不同的资料
3. **分析文献的作者如何处理与相关领域内的其他作者、证据和观点的关系，如何形成自己的文章**
4. **提供描述作者的动机、分析方式以及文章结构的词汇**
5. 如何在阅读和做笔记的过程中结合自己的知识和经验

为什么阅读？
1. 阅读是从**自己的观点**阐释文献，也是解决问题的一部分。
2. 阅读就像对话，很多**对话技巧**可以用在阅读中
	1. 比如，就文章进行提问
	2. 判断文章和自己问题的相关性
	3. 作者观点的证据，以及与前人之间的共性或矛盾
3. 对课题思考越多，对文章的理解就越充分，因此你可以提出很多关键问题。
例子：老师想要看自己工作的优势，于是能很快得浏览相关文献（带着自己的观点和问题，来阅读，来支撑自己的观点），后来发现所有做全球的都要做低通滤波。
（似乎对于大道理也是一样的，很多人生道理的阐释似乎也是可以使用类似的方面）

how to read
1. 把读到的东西用自己的语言写下来，避免后期变成抄几篇论文。
2. 无法集中注意力：带着自己的问题看文献
3. 如何阅读地更快：开始的时候慢下来，先对文章有一个大体的理解。
4. 背景知识的积累
	1. 背景知识的积累属于泛读，因此资料选择不要局限于课题本身，时间允许的情况下广泛阅读
	2. 拿出尽量多的时间读一些综述性文章或者基础课程数据，进行整本书的泛读。在泛读的过程中如果遇到一些问题，不要停下来，简短地做一些标注，继续阅读

how to taking notes
1. 做笔记不是单纯记文字，而是形成自己的理解。最好看完后用自己的话再写笔记的内容
2. **将作者的观点用自己的话表达出来，不要摘抄。**
3. 做笔记的目的是为了给问题提供最好的答案，通过笔记建立文献与你之间的联系。
4. 如何理解作者在表达什么
	1. 将作者观点用自己的话表达出来
	2. 从文章中问作者正在做什么
		1. 比如：在前任研究的基础上，作者的主要目的和动机是什么，与他人的联系是什么
		2. 作者分析的模式是什么
		3. 什么样的结构将文章串联起来形成一个整体

作者的主要动机
1. 同意、捍卫、论证某个观点
2. 提出一个新观点
3. 承认某个观点的有点，但在某些重要方面进行论证
4. 换一个方式表达观点
5. 由于某个观点的问题而抛弃他人的观点
6. 以各种合理的理由拒绝、反驳或驳斥他人的观点
7. 通过更高的视角调和两个不同的观点

作者的结构意图
1. 哪里是概括，哪里是细化
2. 哪些表述或段落属于同一个部分
3. 哪里在承上启下
4. 哪里在总结
5. 哪里偏离主旨
